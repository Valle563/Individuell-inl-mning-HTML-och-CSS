Seminars

Introduction to HTML Structure

Getting Started with HTML – October 17th, 10:00 AM – 11:00 AM, with Sarah Lin. The first seminar introduces the fundamentals of HTML and the structure of a web page. Sarah Lin, a front-end developer at CodeWave with over eight years of experience, will explain how web browsers interpret HTML and how to use essential tags like "html", "head", "body", headings, and paragraphs. Participants will learn to set up their first simple webpage and understand how HTML defines the content and layout of every website.

Understanding HTML Tags & Attributes

Understanding HTML Tags & Attributes – October 17th, 11:30 AM – 12:30 PM, with Alex Carter. This session, presented by Alex Carter from LearnHub, explores the deeper side of HTML elements and their attributes. He will discuss links, images, tables, and lists, explaining how each element contributes to the overall structure of a website. The seminar also covers semantic HTML, showing why tags like "header", "section", and "article" are important for both accessibility and SEO.

Building a Personal Webpage from Scratch

Building a Personal Webpage from Scratch – October 17th, 2:00 PM – 3:30 PM, with Maria Gonzalez. Maria Gonzalez, a freelance UI designer, will lead a hands-on session where participants create their first personal webpage. She will guide attendees through designing a simple one-page website that includes a profile, bio, and photo while focusing on clean, semantic HTML5 structure. The goal of this workshop is to help beginners gain confidence by building a functional, well-structured page from start to finish.

Accessibility Basics in HTML

Accessibility Basics in HTML – October 17th, 4:00 PM – 5:00 PM, with David Nguyen. David Nguyen, an accessibility advocate and web standards contributor, will teach the fundamentals of accessible HTML. This session focuses on how to write inclusive markup that works well with screen readers and assistive technologies. Nguyen will explain best practices such as using alt text, ARIA roles, and correct heading hierarchy to ensure that every visitor can navigate and understand your site easily.

Intro to CSS: Styling the Web

Intro to CSS: Styling the Web – October 18th, 10:00 AM – 11:00 AM, with Jasmine Patel. On the second day, the focus shifts to CSS and how it styles HTML content. Jasmine Patel, a front-end engineer at SoftNet, will explain how to link CSS files to HTML, how selectors work, and how to modify basic properties such as colors, fonts, and spacing. Attendees will learn the difference between inline, internal, and external CSS and start applying styles to their own practice pages.

Selectors, Colors, and Fonts

Selectors, Colors, and Fonts – October 18th, 11:30 AM – 12:30 PM, with Leo Moretti. This seminar, led by CSS expert Leo Moretti, dives into one of the most powerful parts of CSS: selectors. Participants will learn to target specific HTML elements and apply unique styles. Moretti will also cover color theory, the use of HEX and RGB color codes, and how to import and use Google Fonts effectively to create visually appealing pages.

Box Model & Layout Fundamentals

Box Model & Layout Fundamentals – October 18th, 2:00 PM – 3:30 PM, with Fiona Brooks. Fiona Brooks, a front-end mentor and former designer at WebNest, will guide participants through the CSS box model, one of the most essential concepts in modern web design. She will explain how padding, borders, and margins affect spacing and how to manipulate these properties to achieve precise layouts. This session will also show how to use browser developer tools to visualize box model behavior in real time.

Positioning & Display Explained

Positioning & Display Explained – October 18th, 4:00 PM – 5:00 PM, with Thomas Lee. In this late afternoon session, web instructor Thomas Lee will discuss how to control element placement using CSS positioning. The session covers static, relative, absolute, and fixed positioning, along with different display types like block, inline, and flex. Lee will demonstrate practical layout examples and show how small changes in positioning can completely alter a webpage’s design.

Responsive Design with CSS Flexbox

Responsive Design with CSS Flexbox – October 19th, 10:00 AM – 11:30 AM, with Clara Johansson. The final day begins with a focus on responsive design, presented by UX engineer Clara Johansson. Participants will learn how Flexbox simplifies layout design by automatically adjusting elements based on screen size. Johansson will guide attendees through building a responsive navigation bar and a flexible card grid that adapts perfectly to both desktop and mobile devices.

CSS Grid for Modern Layouts

CSS Grid for Modern Layouts – October 19th, 12:00 PM – 1:30 PM, with Nathan Kim. Nathan Kim, a senior front-end developer at DevEdge, will present an in-depth introduction to CSS Grid. This session will explain how to create complex, magazine-style layouts using grid columns, rows, and areas. Kim will also compare Grid with Flexbox, helping participants understand when to use each system for the best results in modern responsive design.

Adding Interactivity with Pseudo-Classes & Transitions

Adding Interactivity with Pseudo-Classes & Transitions – October 19th, 2:00 PM – 3:00 PM, with Priya Shah. Creative coder Priya Shah will show how to add interactive effects to websites using CSS pseudo-classes and transitions. Participants will learn how to create hover effects, focus states, and smooth animations that respond to user interaction. The seminar will demonstrate how these simple techniques can make any webpage feel more dynamic and engaging without the need for JavaScript.

Mini Project: Responsive Portfolio Page

Mini Project: Responsive Portfolio Page – October 19th, 3:30 PM – 5:00 PM, with Ben Harris. The seminar series concludes with a comprehensive project session led by Ben Harris, a front-end instructor at CodeLaunch. In this final workshop, participants will combine all the skills learned over the previous days to build a fully responsive personal portfolio site. The project will include a header, navigation menu, gallery section, and contact form, styled using Flexbox and Grid. By the end of the session, attendees will have a complete, professional-quality portfolio page they can showcase as proof of their new skills.
